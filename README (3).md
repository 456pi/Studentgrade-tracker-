# Grade Tracker

## Overview

Grade Tracker is a simple Java console application that allows users to
enter student grades and generates a summary report.

## Features

-   Accepts multiple grades (0--100)
-   Uses `-1` to finish input
-   Validates grade entries
-   Displays:
    -   Total number of students
    -   Average score
    -   Highest score
    -   Lowest score

## Requirements

-   Java JDK 8 or later

## Compile

``` bash
javac GradeTracker.java
```

## Run

``` bash
java GradeTracker
```

## Example

    Enter grade: 85
    Enter grade: 92
    Enter grade: 78
    Enter grade: -1

    Total Students: 3
    Average Score: 85.00
    Highest Score: 92.0
    Lowest Score: 78.0
